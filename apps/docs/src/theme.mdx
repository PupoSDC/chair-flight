# Theme Overview

We are using Joy ui.

<br />

To avoid getting lost doing too many custom components, the approach taken in
this project is to configure the components via the theme. In this page you can
enjoy a demo of components coming straight from joy UI with out custom team
applied to them.

<br />

If you are contributing to this project, you should feel confortable using any
component coming from joy ui out of the box.

<br />
