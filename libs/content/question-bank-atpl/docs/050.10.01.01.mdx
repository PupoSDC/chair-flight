# Surface observations

## Learning Objectives

1.  **050.10.01.01.01** -- Define "gusts", as given in METARs.
2.  **050.10.01.01.02** -- Distinguish wind given in METARs and wind given by
    the control tower for take-off and landing.
3.  **050.10.01.01.03** -- Define "visibility".
4.  **050.10.01.01.04** -- Describe the meteorological measurement of
    visibility.
5.  **050.10.01.01.05** -- Define "prevailing visibility".
6.  **050.10.01.01.06** -- Define "ground visibility".
7.  **050.10.01.01.07** -- List the units used for visibility (m, km, statute
    mile).
8.  **050.10.01.01.08** -- Define "runway visual range".
9.  **050.10.01.01.09** -- Describe the meteorological measurement of runway
    visual range.
10. **050.10.01.01.10** -- Indicate where the transmissometers/forward-scatter
    meters are placed on the aerodrome.
11. **050.10.01.01.11** -- List the units used for runway visual range (m, ft).
12. **050.10.01.01.12** -- List the different possibilities to transmit
    information to pilots about runway visual range.
13. **050.10.01.01.13** -- Compare ground visibility, prevailing visibility, and
    runway visual range.
14. **050.10.01.01.14** -- Indicate the means of observation of present weather.
15. **050.10.01.01.15** -- Indicate the means of observing clouds for the
    purpose of recording: type, amount, height of base (ceilometers), and top.
16. **050.10.01.01.16** -- State the clouds which are indicated in METAR, TAF
    and SIGMET.
17. **050.10.01.01.17** -- Define "oktas".
18. **050.10.01.01.18** -- Define "cloud base".
19. **050.10.01.01.19** -- Define "ceiling".
20. **050.10.01.01.20** -- Name the unit and the reference level used for
    information about cloud base (ft).
21. **050.10.01.01.21** -- Define "vertical visibility".
22. **050.10.01.01.22** -- Explain briefly how and when vertical visibility is
    measured.
23. **050.10.01.01.23** -- Name the units used for vertical visibility (ft, m).
24. **050.10.01.01.24** -- Indicate the means of observation of air temperature
    (thermometer).
25. **050.10.01.01.25** -- Name the units of relative humidity ( per cent) and
    dew-point temperature (Celsius, Fahrenheit).
