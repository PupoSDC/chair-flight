# Memory

## Learning Objectives

1.  **040.03.01.03.01** -- Explain the link between the types of memory (to
    include sensory, working/short-term and long-term memory).
2.  **040.03.01.03.02** -- Describe the differences between the types of memory
    in terms of capacity and retention time.
3.  **040.03.01.03.03** -- Justify the importance of sensory-store memories in
    processing information.
4.  **040.03.01.03.04** -- State the average maximum number of separate items
    that may be held in working memory (5 Â± 2..
5.  **040.03.01.03.05** -- Stress how interruption can affect short-term/working
    memory.
6.  **040.03.01.03.06** -- Give examples of items that are important for pilots
    to hold in working memory during flight.
7.  **040.03.01.03.07** -- Describe how the capacity of the working-memory store
    may be increased.
8.  **040.03.01.03.08** -- State the subdivisions of long-term memory and give
    examples of their content.
9.  **040.03.01.03.09** -- Explain that skills are kept primarily in the
    long-term memory.
10. **040.03.01.03.10** -- Describe amnesia and how it affects memory.
11. **040.03.01.03.11** -- Name the common problems with both the long- and
    short-term memories and the best methods to try to counteract them.
