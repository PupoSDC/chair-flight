# Alternating current (AC)

## Learning Objectives

1.  **021.09.01.03.01** -- Explain the term "alternating current" (AC), and
    compare its use to DC with regard to complexity.
2.  **021.09.01.03.02** -- Define the term "phase", and explain the basic
    principle of single-phase and three-phase AC.
3.  **021.09.01.03.03** -- State that aircraft can use single-phase or
    three-phase AC.
4.  **021.09.01.03.04** -- Define frequency and state the unit of measurement.
5.  **021.09.01.03.05** -- Define "phase shift" in qualitative terms.
