# Arrival and approach segments

## Learning Objectives

1.  **010.06.04.03.01** -- Name the five standard segments of an instrument
    approach procedure, and state the beginning and end for each of them.
2.  **010.06.04.03.02** -- Describe where an arrival route normally ends.
3.  **010.06.04.03.03** -- State the main task of the initial approach segment.
4.  **010.06.04.03.04** -- Describe the maximum angle of interception between
    the initial approach segment and the intermediate approach segment (provided
    at the intermediate fix) for a precision approach and a non-precision
    approach.
5.  **010.06.04.03.05** -- Describe the main task of the intermediate approach
    segment.
6.  **010.06.04.03.06** -- State the main task of the final approach segment.
7.  **010.06.04.03.07** -- Name the two possible aims of a final approach.
8.  **010.06.04.03.08** -- Explain the term "final approach point" in case of an
    ILS approach.
9.  **010.06.04.03.09** -- State what happens if an ILS glide path (GP) becomes
    inoperative during the approach.
