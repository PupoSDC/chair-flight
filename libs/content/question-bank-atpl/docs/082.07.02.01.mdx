# Static longitudinal, roll and directional stability

## Learning Objectives

1.  **082.07.02.01.01** -- Define static stability; give an example of static
    stability and of static instability.
2.  **082.07.02.01.02** -- Explain the contribution of the main rotor to speed
    stability.
3.  **082.07.02.01.03** -- Describe the influence of the horizontal stabiliser
    on static longitudinal stability.
4.  **082.07.02.01.04** -- Explain the effect of hinge offset on static
    stability.
5.  **082.07.02.01.05** -- Describe the influence of the tail rotor on static
    directional stability.
6.  **082.07.02.01.06** -- Describe the influence of the vertical stabiliser on
    static directional stability.
7.  **082.07.02.01.07** -- Explain the influence of the main rotor on static
    roll stability.
8.  **082.07.02.01.08** -- Describe the influence of the longitudinal position
    of the CG on static longitudinal stability.
