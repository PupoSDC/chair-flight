# Combustion chamber

## Learning Objectives

1.  **021.11.02.03.01** -- Define the purpose of the combustion chamber.
2.  **021.11.02.03.02** -- List the requirements for combustion.
3.  **021.11.02.03.03** -- Describe the working principle of a combustion
    chamber.
4.  **021.11.02.03.04** -- Explain the reason for reducing the airflow axial
    velocity at the combustion chamber inlet (snout).
5.  **021.11.02.03.05** -- State the function of the swirl vanes (swirler).
6.  **021.11.02.03.06** -- State the function of the drain valves.
7.  **021.11.02.03.07** -- Define the terms "primary airflow" and "secondary
    airflow", and explain their purpose.
8.  **021.11.02.03.08** -- Explain the following two mixture ratios: primary
    airflow to fuel; total airflow (within the combustion chamber) to fuel.
9.  **021.11.02.03.09** -- Describe the gas-parameter changes in the combustion
    chamber.
10. **021.11.02.03.10** -- State a typical maximum value of the outlet
    temperature of the combustion chamber.
11. **021.11.02.03.11** -- Describe the following types of combustion chambers
    and state the differences between them: can type; can-annular, cannular or
    turbo-annular; annular; reverse-flow annular.
