# Purpose, operation, components, serviceability

## Learning Objectives

1.  **021.16.06.01.01** -- Explain the purpose of a clutch.
2.  **021.16.06.01.02** -- Describe and explain the operation of a: centrifugal
    clutch; actuated clutch.
3.  **021.16.06.01.03** -- List the typical components of the various clutches.
4.  **021.16.06.01.04** -- Identify the following methods by which clutch
    serviceability can be ascertained: brake-shoe dust; vibration; main-rotor
    run-down time; engine speed at time of main-rotor engagement; belt
    tensioning; start protection in a belt-drive clutch system.
