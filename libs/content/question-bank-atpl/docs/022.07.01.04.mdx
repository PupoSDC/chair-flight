# Failures

## Learning Objectives

1.  **022.07.01.04.01** -- Explain the various redundancies and independent
    systems that are built into the AFCSs.
2.  **022.07.01.04.02** -- Appreciate that the pilot can override the system in
    the event of a failure.
3.  **022.07.01.04.03** -- Explain a series actuator "hard over" which equals
    aircraft attitude runaway.
4.  **022.07.01.04.04** -- Explain the consequences of a saturation of the
    series actuators.
