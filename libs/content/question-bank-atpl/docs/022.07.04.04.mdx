# Flight director: design and operation

## Learning Objectives

1.  **022.07.04.04.01** -- Explain the purpose of a flight director system.
2.  **022.07.04.04.02** -- Describe the different types of display: pitch and
    roll crossbars; V-bar.
3.  **022.07.04.04.03** -- State the difference between the flight director
    system and the autopilot system. Explain how each can be used independently.
4.  **022.07.04.04.04** -- List and describe the main components of the flight
    director system.
5.  **022.07.04.04.05** -- Give examples of different situations with the
    respective indications of the command bars.
6.  **022.07.04.04.06** -- Explain the architecture of the different flight
    directors fitted to helicopters and the importance to monitor other
    instruments as well as the flight director.
7.  **022.07.04.04.07** -- Explain how some helicopter types have the collective
    setting as a flight director command; however, the command does not provide
    protection against a transmission overtorque.
8.  **022.07.04.04.08** -- Describe the collective setting and yaw depiction on
    flight director for some helicopters.
