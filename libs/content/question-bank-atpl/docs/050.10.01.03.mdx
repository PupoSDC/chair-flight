# Satellite observations

## Learning Objectives

1.  **050.10.01.03.01** -- Describe the basic outlines of satellite
    observations.
2.  **050.10.01.03.02** -- Name the main uses of satellite pictures in aviation
    meteorology.
3.  **050.10.01.03.03** -- Describe the different types of satellite imagery.
4.  **050.10.01.03.04** -- Interpret qualitatively the satellite pictures in
    order to get useful information for flights: location of clouds (distinguish
    between stratiform and cumuliform clouds).
5.  **050.10.01.03.05** -- Interpret qualitatively the satellite pictures in
    order to get useful information for flights: location of fronts.
6.  **050.10.01.03.06** -- Interpret qualitatively the satellite pictures in
    order to get useful information for flights using atmospheric motion vector
    images to locate jet streams.
