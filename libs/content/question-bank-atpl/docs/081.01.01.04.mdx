# Shape of an aerofoil section

## Learning Objectives

1.  **081.01.01.04.01** -- Describe the following parameter of an aerofoil
    section: leading edge.
2.  **081.01.01.04.02** -- Describe the following parameter of an aerofoil
    section: trailing edge.
3.  **081.01.01.04.03** -- Describe the following parameter of an aerofoil
    section: chord line.
4.  **081.01.01.04.04** -- Describe the following parameter of an aerofoil
    section: thickness-to-chord ratio or relative thickness.
5.  **081.01.01.04.05** -- Describe the following parameter of an aerofoil
    section: location of maximum thickness.
6.  **081.01.01.04.06** -- Describe the following parameter of an aerofoil
    section: camber line.
7.  **081.01.01.04.07** -- Describe the following parameter of an aerofoil
    section: camber.
8.  **081.01.01.04.08** -- Describe the following parameter of an aerofoil
    section: nose radius.
9.  **081.01.01.04.09** -- Describe a symmetrical and an asymmetrical aerofoil
    section.
