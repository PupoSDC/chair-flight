# AC generation

## Learning Objectives

1.  **021.09.03.02.01** -- Describe the working principle of a brushless
    three-phase AC generator.
2.  **021.09.03.02.02** -- State that the generator field current is used to
    control voltage.
3.  **021.09.03.02.03** -- State the relationship between output frequency and
    the rpm of a three-phase AC generator.
4.  **021.09.03.02.04** -- Explain the term "frequency wild generator".
5.  **021.09.03.02.05** -- List the following different power sources that can
    be used for an aeroplane to drive an AC generator: engine; APU; RAT;
    hydraulic.
6.  **021.09.03.02.06** -- List the following different power sources that can
    be used for a helicopter to drive an AC generator: engine; APU; gearbox.
