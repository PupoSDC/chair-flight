# Principles

## Learning Objectives

1.  **062.02.02.01.01** -- Define the acronym "NDB": non-directional radio
    beacon.
2.  **062.02.02.01.02** -- Define the acronym "ADF": automatic direction-finding
    equipment.
3.  **062.02.02.01.03** -- State that the NDB is the ground part of the system.
4.  **062.02.02.01.04** -- State that the ADF is the airborne part of the
    system.
5.  **062.02.02.01.05** -- State that the NDB operates in the LF and MF
    frequency bands.
6.  **062.02.02.01.06** -- State that the frequency band assigned to
    aeronautical NDBs according to ICAO Annex 10 is 190-1750 kHz.
7.  **062.02.02.01.07** -- Define a "locator beacon": an LF/MF NDB used as an
    aid to final approach usually with a range of 10-25 NM.
8.  **062.02.02.01.08** -- State that certain commercial radio stations transmit
    within the frequency band of the NDB.
9.  **062.02.02.01.09** -- State that according to ICAO Annex 10, an NDB station
    has an automatic ground monitoring system.
10. **062.02.02.01.10** -- Describe the use of NDBs for navigation.
11. **062.02.02.01.11** -- Describe the procedure to identify an NDB station.
12. **062.02.02.01.12** -- Interpret the term "cone of confusion" in respect of
    an NDB.
13. **062.02.02.01.13** -- State that an NDB station emits a N0N/A1A or a
    N0N/A2A signal.
14. **062.02.02.01.14** -- State the function of the beat frequency oscillator
    (BFO).
15. **062.02.02.01.15** -- State that in order to identify a N0N/A1A NDB, the
    BFO circuit of the receiver has to be activated.
16. **062.02.02.01.16** -- State that on modern aircraft, the BFO is activated
    automatically.
