# Automatic guidance (upper modes of AFCS)

## Learning Objectives

1.  **022.07.04.03.01** -- Explain the function of the attitude-hold system in
    an AFCS.
2.  **022.07.04.03.02** -- Explain the function of the heading-hold system in an
    AFCS.
3.  **022.07.04.03.03** -- Explain the function of the vertical-speed hold
    system in an AFCS.
4.  **022.07.04.03.04** -- Explain the function of the navigation-coupling
    system in an AFCS.
5.  **022.07.04.03.05** -- Explain the function of the VOR-/ILS-coupling system
    in an AFCS.
6.  **022.07.04.03.06** -- Explain the function of the hover-mode system in an
    AFCS (including Doppler and radio-altimeter systems).
7.  **022.07.04.03.07** -- Explain the function of the SAR mode (automatic
    transition to hover and back to cruise) in an AFCS.
