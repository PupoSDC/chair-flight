# Ground Direction Finding: Errors and accuracy

## Learning Objectives

1.  **062.02.01.04.01** -- Explain why synchronous transmissions will cause
    errors.
2.  **062.02.01.04.02** -- Describe the effect of "multipath signals".
3.  **062.02.01.04.03** -- Explain that VDF information is divided into the
    following classes according to ICAO Annex 10:
    - Class A: accurate to a range within plus/minus 2 degrees
    - Class B: accurate to a range within plus/minus 5 degrees ;
    - Class C: accurate to a range within plus/minus 10 degrees ;
    - Class D: accurate to less than Class C.
