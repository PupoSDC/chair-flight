# Alerting service

## Learning Objectives

1.  **010.07.01.06.01** -- State who provides the alerting service.
2.  **010.07.01.06.02** -- State who is responsible for initiating the
    appropriate emergency phase.
3.  **010.07.01.06.03** -- State the aircraft to which alerting service shall be
    provided.
4.  **010.07.01.06.04** -- State which unit shall be notified by the responsible
    ATS unit immediately when an aircraft is considered to be in a state of
    emergency.
5.  **010.07.01.06.05** -- Name the three stages of emergency and describe the
    basic conditions for each kind of emergency.
6.  **010.07.01.06.06** -- State the meaning of the expressions "INCERFA",
    "ALERFA" and "DETRESFA".
7.  **010.07.01.06.07** -- State the information to be provided to those
    aircraft that operate in the vicinity of an aircraft that is either in a
    state of emergency or unlawful interference.
