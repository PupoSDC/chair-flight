# Gas turbine engine: bleed-air supply

## Learning Objectives

1.  **021.06.01.02.01** -- State that the possible bleed-air sources for gas
    turbine engine aircraft are the following: engine; auxiliary power unit
    (APU); ground supply.
2.  **021.06.01.02.02** -- State that for an aeroplane a bleed-air supply can be
    used for the following systems or components: ice protection; engine air
    starter; pressurisation of a hydraulic reservoir; air-driven hydraulic
    pumps; pressurisation and air conditioning.
3.  **021.06.01.02.03** -- State that for a helicopter a bleed-air supply can be
    used for the following systems or components: anti-icing; engine air
    starter; pressurisation of a hydraulic reservoir.
4.  **021.06.01.02.04** -- State that the bleed-air supply system can comprise
    the following: pneumatic ducts; isolation valve; pressure-regulating valve;
    engine bleed valve (HP/IP valves); fan-air pre-cooler; temperature and
    pressure sensors.
5.  **021.06.01.02.05** -- Interpret a basic pneumatic system schematic to the
    level of detail as found in an FCOM.
6.  **021.06.01.02.06** -- Describe the cockpit indications for bleed-air
    systems.
7.  **021.06.01.02.07** -- Explain how the bleed-air supply system is controlled
    and monitored.
8.  **021.06.01.02.08** -- State the following bleed-air malfunctions:
    over-temperature; over-pressure; low pressure; overheat/duct leak; and
    describe the potential consequences.
