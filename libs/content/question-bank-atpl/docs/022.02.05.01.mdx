# VSI and instantaneous vertical speed indicator (IVSI)

## Learning Objectives

1.  **022.02.05.01.01** -- List the two units used for VSIs and state the
    relationship between them: metres per second; feet per minute.
2.  **022.02.05.01.02** -- Explain the operating principles of a VSI and an
    IVSI.
3.  **022.02.05.01.03** -- Describe and compare the following types of VSIs:
    barometric type (VSI); instantaneous barometric type (IVSI); inertial type
    (inertial information provided by an inertial reference unit).
4.  **022.02.05.01.04** -- Describe the following VSI errors: static system
    errors; instrument errors; time lag.
5.  **022.02.05.01.05** -- Describe the effects on a VSI of a blockage or a
    leakage on the static pressure line.
6.  **022.02.05.01.06** -- Give examples of a VSI display.
7.  **022.02.05.01.07** -- Compare the indications of a VSI and an IVSI during
    flight in turbulence and appropriate pilot technique during manoeuvring
    using either type.
