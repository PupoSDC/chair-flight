# Spectrum, bands, range

## Learning Objectives

1.  **090.06.01.01.01** -- Describe the radio-frequency spectrum with particular
    reference to VHF.
2.  **090.06.01.01.02** -- Describe the radio-frequency spectrum of the bands
    into which the radio-frequency spectrum is divided.
3.  **090.06.01.01.03** -- Identify the frequency range of the VHF band.
4.  **090.06.01.01.04** -- State the band normally used for aeronautical mobile
    service (AMS) voice communication.
5.  **090.06.01.01.05** -- State the frequency separation allocated between
    consecutive VHF frequencies.
6.  **090.06.01.01.06** -- List the factors which reduce the effective range and
    quality of VHF radio transmissions.
