# Engine handling

## Learning Objectives

1.  **021.10.10.02.01** -- State the correct procedures for setting the engine
    controls when increasing or decreasing power.
2.  **021.10.10.02.02** -- Define the following terms: take-off power; maximum
    continuous power.
3.  **021.10.10.02.03** -- Describe the start problems associated with extreme
    cold weather.
4.  **021.10.10.02.04** -- Describe the principal difference between a
    full-authority digital engine control (FADEC) system-controlled engine and
    traditional manual engine controls.
5.  **021.10.10.02.05** -- Describe the engine controls available on the flight
    deck for a FADEC-controlled engine.
6.  **021.10.10.02.06** -- Explain that the FADEC has full authority of the
    control of all engine parameters ensuring efficient and correct running of
    the engine, including protection in the event of failure.
7.  **021.10.10.02.07** -- Explain the need for FADEC redundancy with regard to
    power supply and data input and output.
