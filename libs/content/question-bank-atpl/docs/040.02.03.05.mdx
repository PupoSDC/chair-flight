# Incapacitation in flight

## Learning Objectives

1.  **040.02.03.05.01** -- State that incapacitation is most dangerous when its
    onset is insidious.
2.  **040.02.03.05.02** -- List the major causes of in-flight incapacitation.
3.  **040.02.03.05.03** -- State the importance of crew to be able to recognise
    and promptly react upon incapacitation of other crew members, should it
    occur in flight.
4.  **040.02.03.05.04** -- Explain methods and procedures to cope with
    incapacitation in flight.
