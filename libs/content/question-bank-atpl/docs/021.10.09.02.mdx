# Constant-speed propeller: design, operation, system components

## Learning Objectives

1.  **021.10.09.02.01** -- Describe the operating principle of a constant-speed
    propeller system under normal flight operations with the aid of a schematic.
2.  **021.10.09.02.02** -- Explain the need for a MAP indicator to control the
    power setting with a constant-speed propeller.
3.  **021.10.09.02.03** -- State the purpose of a torque-meter.
4.  **021.10.09.02.04** -- State the purpose and describe the operation of a
    low-pitch stop (centrifugal latch).
5.  **021.10.09.02.05** -- Describe the operating principle of a single-acting
    and a double-acting variable pitch propeller for single- and multi-engine
    aeroplanes.
6.  **021.10.09.02.06** -- Describe the function and the basic operating
    principle of synchronising and synchro-phasing systems.
7.  **021.10.09.02.07** -- Explain the purpose and the basic operating principle
    of an auto-feathering system and unfeathering.
