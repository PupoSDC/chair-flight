# Types, system components, design, operation, degraded modes of operation, indications and warnings

## Learning Objectives

1.  **021.06.02.01.01** -- Describe the purpose of an air-conditioning system.
2.  **021.06.02.01.02** -- Explain how an air-conditioning system is controlled.
3.  **021.06.02.01.03** -- Describe the vapour cycle air-conditioning system
    including system components, design, operation, degraded modes of operation
    and system malfunction indications.
4.  **021.06.02.01.04** -- Identify the following components from a diagram of
    an air-conditioning system and describe the operating principle and
    function: air-cycle machine (pack, bootstrap system); pack-cooling fan;
    water separator; mixing valves; flow-control valves; isolation valves;
    recirculation fans; filters for recirculation; temperature sensors.
5.  **021.06.02.01.05** -- List and describe the controls, indications and
    warnings related to an air-conditioning system.
