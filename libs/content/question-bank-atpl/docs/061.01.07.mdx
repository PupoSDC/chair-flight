# Dead reckoning (DR)

## Learning Objectives

1.  **061.01.07.01.01** -- Determine a DR position.
2.  **061.01.07.01.02** -- Evaluate the difference between a DR and a fix
    position.
3.  **061.01.07.01.03** -- Define "speed factor" (SF). Speed divided by 60, used
    for mental flight-path calculations.
4.  **061.01.07.01.04** -- Calculate wind correction angle (WCA) using the
    formula: WCA = XWC (crosswind component)/SF
