# System components, design, operation, indications and warnings

## Learning Objectives

1.  **021.04.03.02.01** -- Explain the limitation of brake energy and describe
    the operational consequences.
2.  **021.04.03.02.02** -- Explain how brakes are actuated: hydraulically,
    electrically.
3.  **021.04.03.02.03** -- Explain the purpose of an in-flight wheel brake
    system.
4.  **021.04.03.02.04** -- Describe the function of a brake accumulator.
5.  **021.04.03.02.05** -- Describe the function of the parking brake.
6.  **021.04.03.02.06** -- Explain the function of brake-wear indicators.
7.  **021.04.03.02.07** -- Explain the reason for the brake-temperature
    indicator.
