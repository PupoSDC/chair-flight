# Take-off and landing (including hover)

## Learning Objectives

1.  **034.02.02.01.01** -- Explain the take-off and landing requirements.
2.  **034.02.02.01.02** -- Explain the maximum allowed take-off and landing
    mass.
3.  **034.02.02.01.03** -- Explain that mass has to be restricted to HIGE.
4.  **034.02.02.01.04** -- Explain that if HIGE is unlikely to be achieved (for
    example, blocked by an obstruction), then mass must be restricted to HOGE.
