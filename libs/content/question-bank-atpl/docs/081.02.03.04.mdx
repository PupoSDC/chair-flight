# Effect on pitching moment

## Learning Objectives

1.  **081.02.03.04.01** -- Discuss the effect of Mach number on the CP location.
2.  **081.02.03.04.02** -- Describe the overall change in pitching moment above
    MCRIT and explain the "tuck under" or "Mach tuck" effect.
3.  **081.02.03.04.03** -- State the requirement for a Mach trim system to
    compensate for the effect of the CP movement and "tuck under" effect.
4.  **081.02.03.04.04** -- Discuss the aerodynamic functioning of the Mach trim
    system.
5.  **081.02.03.04.05** -- Discuss the corrective measures if the Mach trim
    fails.
