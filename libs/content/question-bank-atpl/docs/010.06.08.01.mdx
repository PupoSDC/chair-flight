# Operation of transponders

## Learning Objectives

1.  **010.06.08.01.01** -- State when and where the pilot shall operate the
    transponder.
2.  **010.06.08.01.02** -- State the modes and codes that the pilot shall
    operate in the absence of any ATC directions or regional air navigation
    agreements.
3.  **010.06.08.01.03** -- State when the pilot shall operate Mode C.
4.  **010.06.08.01.04** -- State when the pilot shall "SQUAWK IDENT".
5.  **010.06.08.01.05** -- State the transponder code to indicate: a state of
    emergency; a COM failure; unlawful interference.
6.  **010.06.08.01.06** -- Describe the consequences of a transponder failure in
    flight.
7.  **010.06.08.01.07** -- State the primary action of the pilot in the case of
    an unserviceable transponder before departure when no repair or replacement
    at the given AD is possible.
8.  **010.06.08.01.08** -- State when the pilot shall operate Mode S.
