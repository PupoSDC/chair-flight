# PBN point-in-space (PinS) approach

## Learning Objectives

1.  **062.07.05.09.01** -- State that a PinS approach procedure is an instrument
    RNP APCH procedure designed for helicopters only, and that it may be
    published with LNAV minima or LPV minima.
2.  **062.07.05.09.02** -- State that a PinS approach procedure includes either
    a "proceed VFR" or a "proceed visually" instruction from the missed approach
    point (MAPt) to a landing location.
3.  **062.07.05.09.03** -- Recognise the differences between "proceed VFR" and
    "proceed visually".
