# Fully powered controls (irreversible)

## Learning Objectives

1.  **021.05.01.03.01** -- Explain the basic principle of a fully powered
    control system.
2.  **021.05.01.03.02** -- Explain the concept of irreversibility in a flight
    control system.
3.  **021.05.01.03.03** -- Explain the need for a "feel system" in a fully
    powered control system.
4.  **021.05.01.03.04** -- Explain the operating principle of a stabiliser trim
    system in a fully powered control system.
5.  **021.05.01.03.05** -- Explain the operating principle of rudder and aileron
    trim in a fully powered control system.
