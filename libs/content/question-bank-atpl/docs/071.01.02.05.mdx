# All-weather operations

## Learning Objectives

1.  **071.01.02.05.01** -- Explain the operator"s responsibility regarding
    aerodrome/heliport operating minima.
2.  **071.01.02.05.02** -- Define the following terms: "circling",
    "low-visibility procedures", "low-visibility take-off", "visual approach".
3.  **071.01.02.05.03** -- Define the following terms: "flight control system",
    "fail-passive flight control system", "fail-operational flight control
    system", "fail-operational hybrid landing system".
4.  **071.01.02.05.04** -- Define the following terms: "final approach and
    take-off area".
5.  **071.01.02.05.05** -- Explain the general operating requirements for
    low-visibility operations.
6.  **071.01.02.05.06** -- Define aerodrome/heliport considerations regarding
    low-visibility operations.
7.  **071.01.02.05.07** -- Explain the training and qualification requirements
    for flight crew to conduct low-visibility operations.
8.  **071.01.02.05.08** -- Explain the operating procedures for low-visibility
    operations.
9.  **071.01.02.05.09** -- Explain the operator"s and commander"s
    responsibilities regarding minimum equipment for low-visibility operations.
10. **071.01.02.05.10** -- Explain the VFR operating minima.
11. **071.01.02.05.11** -- Aerodrome operating minima: explain under which
    conditions the commander can commence take-off.
12. **071.01.02.05.12** -- Aerodrome operating minima: explain that take-off
    minima are expressed as visibility or runway visual range (RVR).
13. **071.01.02.05.13** -- Aerodrome operating minima: explain the take-off RVR
    value depending on the aerodrome facilities.
14. **071.01.02.05.14** -- Aerodrome operating minima: explain the system minima
    for non-precision approach (NPA) (minimum descent altitude/height (MDA/H)
    and decision altitude/height (DA/H), not RVR).
15. **071.01.02.05.15** -- Aerodrome operating minima: explain under which
    conditions a pilot can continue the approach below MDA/H or DA/H.
16. **071.01.02.05.16** -- Aerodrome operating minima: explain the lowest minima
    for precision approach category 1 (including single-pilot operations).
17. **071.01.02.05.17** -- Aerodrome operating minima: explain the lowest minima
    for precision approach category 2 operations.
18. **071.01.02.05.18** -- Aerodrome operating minima: explain the lowest minima
    for precision approach category 3 operations.
19. **071.01.02.05.19** -- Aerodrome operating minima: explain the lowest minima
    for circling and visual approach.
20. **071.01.02.05.20** -- Aerodrome operating minima: explain the RVR value and
    cloud ceiling depending on the aerodrome.
21. **071.01.02.05.21** -- Aerodrome operating minima: explain under which
    conditions an airborne radar approach can be performed and state the
    relevant minima.
