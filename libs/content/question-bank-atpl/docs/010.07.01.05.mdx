# Flight information service (FIS)

## Learning Objectives

1.  **010.07.01.05.01** -- State for which aircraft FIS shall be provided.
2.  **010.07.01.05.02** -- State whether or not FIS shall include the provision
    of pertinent significant meteorological information (SIGMET) and air
    meteorological information report (AIRMET) information.
3.  **010.07.01.05.03** -- State which information FIS shall include in addition
    to SIGMET and AIRMET information.
4.  **010.07.01.05.04** -- Indicate which other information the FIS shall
    include in addition to the special information given in Annex 11.
5.  **010.07.01.05.05** -- State the meaning of the acronym "ATIS" in plain
    language.
6.  **010.07.01.05.06** -- List the basic information concerning automatic
    terminal information service (ATIS) broadcasts (e.g. frequencies used,
    number of ADs included, updating, identification, acknowledgment of receipt,
    language and channels, ALT- setting).
7.  **010.07.01.05.07** -- State the content of an ATIS message.
8.  **010.07.01.05.08** -- State the reasons and circumstances when an ATIS
    message shall be updated.
