# Distress

## Learning Objectives

1.  **090.05.01.01.01** -- State the DISTRESS signal(s) and DISTRESS
    procedure(s).
2.  **090.05.01.01.02** -- Define "DISTRESS".
3.  **090.05.01.01.03** -- Identify the frequencies that should be used by
    aircraft in DISTRESS.
4.  **090.05.01.01.04** -- Specify the emergency SSR codes that may be used by
    aircraft, and the meaning of the codes.
5.  **090.05.01.01.05** -- Describe the action to be taken by the station which
    receives a DISTRESS message.
6.  **090.05.01.01.06** -- Describe the action to be taken by all other stations
    when a DISTRESS procedure is in progress.
7.  **090.05.01.01.07** -- List the correctly sequenced elements of a DISTRESS
    signal/message and describe the message content.
8.  **090.05.01.01.08** -- Describe the use of discrete frequencies (DEF) in
    case of distress or urgency.
9.  **090.05.01.01.09** -- State that DISTRESS messages take priority over all
    other messages.
