# Position reference system

## Learning Objectives

1.  **061.01.02.01.01** -- State that geodetic latitude and longitude is used to
    define a position on the WGS-84 ellipsoid.
2.  **061.01.02.01.02** -- Define geographic (geodetic) latitude and parallels
    of latitude.
3.  **061.01.02.01.03** -- Calculate the difference in latitude between any two
    given positions.
4.  **061.01.02.01.04** -- Define geographic (geodetic) longitude and meridians.
5.  **061.01.02.01.05** -- Calculate the difference in longitude between any two
    given positions.
