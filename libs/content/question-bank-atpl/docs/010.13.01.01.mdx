# Definitions and descriptions

## Learning Objectives

1.  **010.13.01.01.01** -- Recall the definitions of the following terms:
    accident, aircraft, flight recorder, incident, investigation, maximum mass,
    operator, serious incident, serious injury, State of Design, State of
    Manufacture, State of Occurrence, State of the Operator, State of Registry.
2.  **010.13.01.01.02** -- Explain the difference between "serious incident" and
    "accident".
3.  **010.13.01.01.03** -- Determine whether a certain occurrence has to be
    defined as a serious incident or as an accident.
4.  **010.13.01.01.04** -- Recognise the description of an accident or incident.
