# Units, terms, types, operating principles, displays, errors, corrections

## Learning Objectives

1.  **022.02.04.01.01** -- List the following two units used for altimeters and
    state the relationship between them: feet; metres.
2.  **022.02.04.01.02** -- Define the following terms: height, altitude;
    indicated altitude, true altitude; pressure altitude, density altitude.
3.  **022.02.04.01.03** -- Define the following barometric references: "QNH",
    "QFE", "1013.25".
4.  **022.02.04.01.04** -- Explain the operating principles of an altimeter.
5.  **022.02.04.01.05** -- Describe and compare the following three types of
    altimeters and reason(s) why particular designs may be required in certain
    airspace:
    - simple altimeter (single capsule);
    - sensitive altimeter (multi-capsule);
    - servo-assisted altimeter.
6.  **022.02.04.01.06** -- Give examples of associated displays: pointer,
    multi-pointer, drum, vertical straight scale.
7.  **022.02.04.01.07** -- Describe the following errors: static system error;
    instrument error; barometric error; temperature error (air column not at ISA
    conditions); lag (altimeter response to change of height).
8.  **022.02.04.01.08** -- Demonstrate the use of an altimeter correction table
    for the following errors: temperature corrections; aircraft position errors.
9.  **022.02.04.01.09** -- Describe the effects of a blockage or a leakage on
    the static pressure line.
10. **022.02.04.01.10** -- Describe the use of GPS altitude as an alternative
    means of checking erroneous altimeter indications, and highlight the
    limitations of the GPS altitude indication.
