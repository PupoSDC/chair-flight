# Position reporting

## Learning Objectives

1.  **010.07.02.10.01** -- Describe when position reports shall be made by an
    aircraft flying on routes defined by designated significant points.
2.  **010.07.02.10.02** -- List the six items that are normally included in a
    voice position report.
3.  **010.07.02.10.03** -- State the requirements for using a simplified
    position report with FL, next position (and time-over) and ensuing
    significant points omitted.
4.  **010.07.02.10.04** -- State the item of a position report which must be
    forwarded on to ATC with the initial call after changing to a new frequency.
5.  **010.07.02.10.05** -- Indicate the item of a position report which may be
    omitted if secondary surveillance radar (SSR) Mode C is used.
6.  **010.07.02.10.06** -- Explain in which circumstances the airspeed should be
    included in a position report.
7.  **010.07.02.10.07** -- Explain the meaning of the acronym "ADS".
8.  **010.07.02.10.08** -- Describe which expression shall precede the level
    figures in a position report if the level is reported in relation to 1013.2
    hPa (standard pressure).
