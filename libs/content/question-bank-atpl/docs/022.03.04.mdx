# Flux valve

## Learning Objectives

1.  **022.03.04.01.01** -- Explain the purpose of a flux valve.
2.  **022.03.04.01.02** -- Explain its operating principle.
3.  **022.03.04.01.03** -- Indicate typical locations of the flux valve(s).
4.  **022.03.04.01.04** -- Give the remote-reading compass system as example of
    application for a flux valve.
5.  **022.03.04.01.05** -- Explain that deviation is compensated for and,
    therefore, eliminates the need for a deviation correction card.
6.  **022.03.04.01.06** -- Explain that a flux valve does not suffer from the
    same magnitude of errors as a direct-reading magnetic compass when turning,
    accelerating or decelerating and during pitch-up or pitch-down manoeuvres.
