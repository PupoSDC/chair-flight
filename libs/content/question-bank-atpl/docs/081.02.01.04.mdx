# Compressibility

## Learning Objectives

1.  **081.02.01.04.01** -- State that compressibility means that density can
    change along a streamline, and that this occurs in the high subsonic (from
    Mach 0.4., transonic, and supersonic flow.
2.  **081.02.01.04.02** -- State that compressibility negatively affects the
    pressure gradient, leading to an overall reduction of the CL.
3.  **081.02.01.04.03** -- State that Mach number is a measure of
    compressibility.
4.  **081.02.01.04.04** -- Describe that compressibility increases low-speed
    stall speed and decreases alphaCRIT.
