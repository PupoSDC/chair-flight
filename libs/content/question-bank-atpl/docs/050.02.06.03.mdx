# Clear-air turbulence (CAT) - description, cause and location

## Learning Objectives

1.  **050.02.06.03.01** -- Describe CAT.
2.  **050.02.06.03.02** -- Describe the formation of CAT.
3.  **050.02.06.03.03** -- State where CAT is found in association with jet
    streams, in high-level troughs and in other disturbed high-level air flows.
    (Refer to Subject 050 09 02 02.
4.  **050.02.06.03.04** -- State that remote sensing of CAT from satellites is
    not possible and that forecasting is limited.
5.  **050.02.06.03.05** -- State that pilot reports of turbulence are a very
    valuable source of information as remote measurements are not available.
