# Aeroplane: exhaust

## Learning Objectives

1.  **021.11.02.05.01** -- Name the following main components of the exhaust
    unit and their function:
    - jet pipe;
    - propelling nozzle;
    - exhaust cone.
2.  **021.11.02.05.02** -- Describe the working principle of the exhaust unit.
3.  **021.11.02.05.03** -- Describe the gas-parameter changes in the exhaust
    unit.
4.  **021.11.02.05.04** -- Define the term "choked exhaust nozzle" (not
    applicable to turboprops).
5.  **021.11.02.05.05** -- Explain how jet exhaust noise can be reduced.
