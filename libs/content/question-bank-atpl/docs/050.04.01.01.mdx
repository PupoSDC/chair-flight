# Cloud formation

## Learning Objectives

1.  **050.04.01.01.01** -- Explain cloud formation by adiabatic cooling,
    conduction, advection and radiation.
2.  **050.04.01.01.02** -- Describe cloud formation based on the following
    lifting processes: unorganised lifting in thin layers and turbulent mixing;
    forced lifting at fronts or over mountains; free convection.
3.  **050.04.01.01.03** -- List cloud types typical for stable and unstable air
    conditions.
4.  **050.04.01.01.04** -- Summarise the conditions for the dissipation of
    clouds.
