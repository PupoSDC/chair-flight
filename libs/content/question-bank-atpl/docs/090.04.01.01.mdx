# Action required to be taken in case of communication failure

## Learning Objectives

1.  **090.04.01.01.01** -- State the action to be taken in case of communication
    failure on a controlled VFR flight.
2.  **090.04.01.01.02** -- Identify the frequencies to be used in an attempt to
    establish communication.
3.  **090.04.01.01.03** -- State the additional information that should be
    transmitted in the event of receiver failure.
4.  **090.04.01.01.04** -- Identify the SSR code that may be used to indicate
    communication failure.
5.  **090.04.01.01.05** -- Explain the action to be taken by a pilot that
    experiences a communication failure in the aerodrome traffic pattern at
    controlled aerodromes.
6.  **090.04.01.01.06** -- Describe the action to be taken in case of
    communication failure on an IFR flight.
7.  **090.04.01.01.07** -- Describe the action to be taken in case of
    communication failure on an IFR flight when flying in visual meteorological
    conditions (VMC) and the flight will be terminated in VMC.
8.  **090.04.01.01.08** -- Describe the action to be taken in case of
    communication failure on an IFR flight when flying in instrument
    meteorological conditions (IMC).
9.  **090.04.01.01.09** -- Explain the causes and possible safety impacts of a
    blocked frequency.
