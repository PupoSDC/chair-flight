# Planning courses, distances and cruising levels with VFR charts

## Learning Objectives

1.  **033.01.01.02.01** -- Choose visual waypoints in accordance with specified
    criteria (large, unique, contrast, vertical extent, etc.).
2.  **033.01.01.02.02** -- Measure courses and distances from a VFR chart.
3.  **033.01.01.02.03** -- Find the highest obstacle within a given distance on
    either side of the course.
4.  **033.01.01.02.04** -- Find the following data from a VFR chart and transfer
    them to a navigation plan: waypoints or turning points; distances;
    true/magnetic courses.
5.  **033.01.01.02.05** -- Calculate the minimum pressure altitude with a given
    obstacle clearance or true altitude from a given altitude or pressure
    altitude from minimum grid-area altitude using outside air temperature (OAT)
    and QNH.
6.  **033.01.01.02.06** -- Calculate the vertical or horizontal distance and
    time to climb or descend to/from a given level or altitude with given data.
7.  **033.01.01.02.07** -- Explain how to determine the position of a
    significant VFR point for insertion into a global navigation satellite
    system (GNSS) flight plan, using the distance and bearing from an existing
    significant point and using coordinates.
