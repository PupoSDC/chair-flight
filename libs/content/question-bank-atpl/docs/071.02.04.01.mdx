# Noise-abatement procedures

## Learning Objectives

1.  **071.02.04.01.01** -- Define the operator"s responsibilities regarding the
    establishment of noise-abatement procedures.
2.  **071.02.04.01.02** -- State the main purpose of noise-abatement departure
    procedure (NADP) 1 and NADP 2.
3.  **071.02.04.01.03** -- State that the PIC/commander has the authority to
    decide not to execute an NADP if conditions preclude the safe execution of
    the procedure.
