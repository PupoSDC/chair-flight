# Landing (performance data)

## Learning Objectives

1.  **032.05.03.01.01** -- Determine the field length required for landing with
    a given landing mass from the aeroplane performance data sheets.
2.  **032.05.03.01.02** -- Determine the landing and approach climb-limited
    landing mass from the aeroplane performance data sheets.
3.  **032.05.03.01.03** -- Calculate the maximum allowable landing mass as the
    lowest of: approach-climb- and landing-climb-limited landing mass;
    landing-field-length-limited landing mass; structural-limited landing mass.
4.  **032.05.03.01.04** -- Determine the brake cooling time for different
    landing masses using the aeroplane performance data sheets.
