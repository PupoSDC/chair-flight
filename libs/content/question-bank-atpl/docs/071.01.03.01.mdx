# Flight management

## Learning Objectives

1.  **071.01.03.01.01** -- Minimum time routes: define and interpret minimum
    time route (route that gives the shortest flight time from departure to
    destination adhering to all ATC and airspace restrictions).
2.  **071.01.03.01.02** -- State the circumstances in which a take-off alternate
    must be selected.
3.  **071.01.03.01.03** -- State the maximum flight distance of a take-off
    alternate for: two-engined aeroplanes; ETOPS-approved aeroplanes; three- or
    four-engined aeroplanes.
4.  **071.01.03.01.04** -- State the factors to be considered in the selection
    of a take-off alternate.
5.  **071.01.03.01.05** -- State when a destination alternate need not be
    selected.
6.  **071.01.03.01.06** -- State when two destination alternates must be
    selected.
7.  **071.01.03.01.07** -- State the factors to be considered in the selection
    of a destination alternate aerodrome.
8.  **071.01.03.01.08** -- State the factors to be considered in the selection
    of an en-route alternate aerodrome.
