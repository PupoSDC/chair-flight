# Interception of civil aircraft - SERA

## Learning Objectives

1.  **010.05.06.01.01** -- List the circumstances in which interception of a
    civil aircraft may occur.
2.  **010.05.06.01.02** -- State what primary action should be carried out by an
    intercepted aircraft.
3.  **010.05.06.01.03** -- State which frequency should primarily be tried in
    order to contact an intercepting aircraft.
4.  **010.05.06.01.04** -- State on which mode and code a transponder on board
    the intercepted aircraft should be operated.
