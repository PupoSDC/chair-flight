# Wing shape

## Learning Objectives

1.  **081.01.01.05.01** -- Describe the following parameter of a wing: span.
2.  **081.01.01.05.02** -- Describe the following parameter of a wing: tip and
    root chord.
3.  **081.01.01.05.03** -- Describe the following parameter of a wing: taper
    ratio.
4.  **081.01.01.05.04** -- Describe the following parameter of a wing: wing
    area.
5.  **081.01.01.05.05** -- Describe the following parameter of a wing: wing
    planform.
6.  **081.01.01.05.06** -- Describe the following parameter of a wing: mean
    geometric chord.
7.  **081.01.01.05.07** -- Describe the following parameter of a wing: mean
    aerodynamic chord (MAC).
8.  **081.01.01.05.08** -- Describe the following parameter of a wing: aspect
    ratio.
9.  **081.01.01.05.09** -- Describe the following parameter of a wing: dihedral
    angle.
10. **081.01.01.05.10** -- Describe the following parameter of a wing: sweep
    angle.
11. **081.01.01.05.11** -- Describe the following parameter of a wing: wing
    twist, geometric and aerodynamic.
12. **081.01.01.05.12** -- Describe the following parameter of a wing: angle of
    incidence. Remark: In certain textbooks, angle of incidence is used as angle
    of attack (alpha). For Part-FCL theoretical knowledge examination purposes,
    this use is discontinued, and the angle of incidence is defined as the angle
    between the aeroplane longitudinal axis and the wing-root chord line.
