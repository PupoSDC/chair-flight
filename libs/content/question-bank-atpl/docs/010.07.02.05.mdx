# ATC clearances

## Learning Objectives

1.  **010.07.02.05.01** -- State which information the issue of an ATC clearance
    is based on.
2.  **010.07.02.05.02** -- Describe what a PIC should do if an ATC clearance is
    not suitable.
3.  **010.07.02.05.03** -- State who bears the responsibility for adhering to
    the applicable rules and regulations whilst flying under the control of an
    ATC unit.
4.  **010.07.02.05.04** -- State the two primary purposes of clearances issued
    by ATC units.
5.  **010.07.02.05.05** -- State why clearances must be issued "early enough" to
    aircraft.
6.  **010.07.02.05.06** -- Explain what is meant by the expression "clearance
    limit".
7.  **010.07.02.05.07** -- Explain the meaning of the phrases "cleared via
    flight planned route", "cleared via (designation) departure" and "cleared
    via (designation) arrival" in an ATC clearance.
8.  **010.07.02.05.08** -- List which items of an ATC clearance shall always be
    read back by the flight crew.
