# Sensor types, operating principles, ice protection, displays, incorrect indications

## Learning Objectives

1.  **022.02.03.01.01** -- Describe the following two types of AoA sensors:
    null-seeking (slotted) probe; vane detector.
2.  **022.02.03.01.02** -- For each type, explain the operating principles.
3.  **022.02.03.01.03** -- Explain how both types are protected against ice.
4.  **022.02.03.01.04** -- Give examples of systems that use the AoA as an
    input, such as: air-data computer; stall warning systems; flight-envelope
    protection systems.
5.  **022.02.03.01.05** -- Give examples of and interpret different types of AoA
    displays: simple light arrays of green, amber and red lights; gauges showing
    a numerical scale.
6.  **022.02.03.01.06** -- Explain the implications for the pilot if the AoA
    indication becomes incorrect but still provides data, e.g. if the sensor is
    frozen in a fixed position.
7.  **022.02.03.01.07** -- Explain how an incorrect AoA measurement can affect
    the controllability of an aircraft with flight-envelope protection.
