# Track and heading

## Learning Objectives

1.  **061.01.03.02.01** -- Calculate XWC by: trigonometry; and MDR.
2.  **061.01.03.02.02** -- Explain and apply the concepts of drift and WCA.
3.  **061.01.03.02.03** -- Calculate the actual track with appropriate data of
    heading and drift.
4.  **061.01.03.02.04** -- Calculate TKE with appropriate data of WCA and drift.
5.  **061.01.03.02.05** -- Calculate the heading change at an off-course fix to
    directly reach the next waypoint using the 1:60 rule.
6.  **061.01.03.02.06** -- Calculate the average drift angle based upon an
    off-course fix observation.
