# Normal shock waves

## Learning Objectives

1.  **081.02.02.02.01** -- Describe a normal shock wave with respect to changes
    in: static temperature; static and total pressure; velocity; local speed of
    sound; Mach number; density.
2.  **081.02.02.02.02** -- Describe a normal shock wave with respect to
    orientation relative to the wing surface.
3.  **081.02.02.02.03** -- Explain the influence of increasing Mach number on a
    normal shock wave, at positive lift, with respect to: strength; position
    relative to the wing; second shock wave at the lower surface.
4.  **081.02.02.02.04** -- Explain the influence of alpha on shock-wave
    intensity and shock-wave location at constant Mach number.
