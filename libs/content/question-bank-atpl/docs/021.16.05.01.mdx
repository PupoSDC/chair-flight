# Lubrication, gearing

## Learning Objectives

1.  **021.16.05.01.01** -- Explain and describe the various arrangements when
    the drive changes direction and the need for an intermediate or tail
    gearbox.
2.  **021.16.05.01.02** -- Explain the lubrication requirements for intermediate
    and tail-rotor gearboxes and methods of checking levels.
3.  **021.16.05.01.03** -- Explain how on most helicopters the tail-rotor
    gearbox contains gearing, etc., for the tail-rotor pitch-change mechanism.
