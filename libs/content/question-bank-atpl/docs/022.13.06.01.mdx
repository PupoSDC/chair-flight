# Purpose, certification, malfunctions

## Learning Objectives

1.  **022.13.06.01.01** -- Explain the purpose of the EFB and list typical
    equipment: computer laptop; tablet device; integrated avionics suite in the
    aircraft.
2.  **022.13.06.01.02** -- Describe the "class" hardware certification:
    portable: portable electronic device (PED) that can be used inside or
    outside the aircraft, is not part of the certified aircraft configuration
    and does not require tools to remove it from the flight-deck cradle, if one
    exists; installed: an electronic device that is considered an aircraft part
    covered by the aircraft airworthiness approval, thus is a minimum equipment
    list (MEL) item in the event of failure.
3.  **022.13.06.01.03** -- Describe the "type" software certification: type A:
    applications whose misuse or malfunctions have no adverse effect on flight
    safety; type B: applications for which evaluation of the hazards presented
    by misuse or malfunctions is required.
4.  **022.13.06.01.04** -- Explain implications of malfunctions with the EFB
    installation in a fully electronic flight-deck environment: mass and balance
    calculations; performance calculations; access to charts; access to manuals.
