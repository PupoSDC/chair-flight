# Monitoring of track and time

## Learning Objectives

1.  **033.06.01.01.01** -- State the reasons for possible deviations from the
    planned track and planned timings.
2.  **033.06.01.01.02** -- Calculate GS by using actual in-flight parameters.
3.  **033.06.01.01.03** -- Calculate the expected leg times by using actual
    in-flight parameters.
4.  **033.06.01.01.04** -- Enter, in the progress of flight, at the checkpoint
    or turning point, the "actual time-over" and the "estimated time-over" for
    the next checkpoint into the flight plan.
5.  **033.06.01.01.05** -- State that it is necessary to determine the position
    of the aircraft accurately before commencing descent in order to ensure safe
    ground clearance.
6.  **033.06.01.01.06** -- Calculate revised ETA based on changes to the
    pre-flight plan, including changes of W/V, cruise level, OAT, distances,
    Mach number and calibrated airspeed (CAS).
