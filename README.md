# Chair Flight

An open source ATPL knowledge base and question bank made by students for
students.

Built with `React`, `joy-ui`, `Next`, `Vite`, and `mdx`.
Managed with `pnpm`, `fnm` and `nx`.

## Installing and running

```
fnm use
pnpm install
pnpm run dev
```
